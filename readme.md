# Go Snakes

## Setup

### Install Go

Ubuntu

`apt install golang-go`

OSX

`brew install go`

### Clone this repository

`go get -v github.com/wmurray8989/go-snakes`

### Install Dependencies

Ubuntu

`apt install libsdl2-dev`
`apt install libsdl2_gfx-dev`

OSX

`brew install pkg-config`
`brew install sdl2`
`brew install sdl2_gfx`

## Run

`go run main.go`
